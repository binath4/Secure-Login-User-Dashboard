# Secure Login & User Dashboard (PHP + MySQL)

A simple authentication system built using PHP and MySQL.

-----------------

##  Project Overview

- User Registration
- Secure Password Hashing
- Login Authentication
- Session Management
- Protected Dashboard
- Logout Functionality

It is designed as a progression project.

## Demonstrated things & Security Features

- Password hashing using `PASSWORD_DEFAULT`

- Protected routes
- PHP Sessions
- Password Security (`password_hash`, `password_verify`)
- MySQL Database Integration
- Basic Access Control

## Instructions

1. Start Apache and MySQL (XAMPP).
2. Create a database named:

   user_db

3. Import the `setup.sql` file.
4. Place project inside:

   C:\xampp\htdocs\"given name"

5. Open in browser:

   http://localhost/given-name
